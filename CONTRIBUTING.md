# Contributing

This is a personal project, but contributions are welcome.

## Development setup

```bash
npm install
```

No build step needed for development. Run commands directly with:

```bash
npx tsx src/cli.ts <command>
```

## Running tests

```bash
npm test
```

Tests load the full 16MB prototype JSON, so expect ~3s startup time.

## Prototype data

The `data/helmod-web-prototypes.json` file is generated by the Factorio mod in `export-mod/`. See the README for regeneration instructions. The checked-in copy is from a Pyanodon modpack save.

## Pull requests

- Keep changes focused — one feature or fix per PR
- Run `npm test` before submitting
- If adding solver features, include test cases that exercise the new behavior
